## Prompt Design Notes

| Decision | Rationale (from coursework) |
|----------|----------------------------|
| **Dense paragraph format** | Per `23_Prompt_Bank.md` — detailed descriptions produce more consistent results |
| **Locked character description** | Per `16_Nano_banana.md` — lock character design and reuse verbatim |
| **Locked background** | Per reference video analysis — anchor scenes all use identical setting |
| **"No extra characters"** | Prevents AI from adding random people in anchor shots |
| **"Space for text overlays"** | Ensures clean areas for CapCut text in final assembly |
| **Same style suffix** | Ensures consistent art style across all 6 images |
| **B-roll uses same mother** | For character cohesion, though clothing/setting can differ |
| **Each B-roll has different child** | Matches reference video pattern (varied children) |
| **Specific lighting per scene** | Anchor = lamplight, B1 = window daylight, B2 = evening lamp, B3 = morning door light |
