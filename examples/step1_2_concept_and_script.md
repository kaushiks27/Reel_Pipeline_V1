## Vibe Brainstorm

**Overall feel**: This video should feel like a wise, warm friend gently sharing a parenting secret over coffee — not lecturing. The reference video's grandmother has that "sage elder" energy. We want the same warmth but younger and culturally adapted.

| Aspect | Direction |
|--------|-----------|
| **Emotional register** | Reassuring, never preachy. The viewer should think "yes, I should do this" not "I'm failing" |
| **Pacing feel** | Gentle rhythm, like soft breathing — each rule lands, pauses, then the next one flows in |
| **Color mood** | Golden warmth (lamplight, sunlight) — like late afternoon. No cold blues or harsh whites |
| **Visual metaphor** | Every B-roll shows physical closeness (kneeling, sitting together, hugging) — the visual language IS the message: be present with your child |
| **Sound feeling** | Like a lullaby with words — the voice should feel like it could put you at ease |
| **What it should NOT feel like** | Corporate training, classroom lecture, guilt-tripping, clinical parenting advice |

## Narration Script (Step 2)

> **Voice blueprint** (copy-paste into EVERY voice prompt — NEVER modify between generations):
> `"subtle Filipino accent, warm maternal tone, medium pace, gentle authority. brief pauses before key phrases as if thinking. sentences taper off slightly with reassurance."`

### Script — Refined Version

```
[HOOK — 4s]
"Every child needs to hear... these three things... every single day."

[RULE 1 — 6s]
"Number one... 'I believe in you.' ...This builds... confidence."

[RULE 2 — 7s]
"Number two... 'Tell me about your day.' ...This creates... connection."

[RULE 3 — 7s]
"Number three... 'I'm proud of you.' ...This teaches... self-worth."

[CTA — 3s]
"Share this... to help another parent."
```

### Refinement Notes
- **Hook tested**: "Every child needs to hear" is stronger than alternatives like "Parents, listen up" or "Don't forget to say" because it leads with the child's perspective, creating empathy.
- **Rule phrasing**: Each rule follows the exact same structure: `"Number [X]... '[phrase].' ...This [verb]... [keyword]."` — this rhythmic consistency makes it feel polished and memorable.
- **Pause placement**: `...` placed before the quoted phrase (builds anticipation), after it (lets it land), and before the keyword (emphasizes the takeaway). Mirrors the reference video's pacing.
- **Word count check**: 55 words total. At ~2 words/second with pauses, this fits comfortably in 27 seconds — leaving 3 seconds for the end card.
- **No words were changed from v1** — the original script was already tight after applying the coursework rules. The refinement pass confirmed quality.
