## Prompt Design Notes

| Decision | Rationale |
|----------|-----------|
| **Locked character verbatim** | Per master directive — character description copy-pasted exactly |
| **Locked background verbatim** | Per master directive — anchor scenes all use identical setting |
| **"No extra characters"** | Prevents AI from adding random people in anchor shots |
| **"Space for text overlays"** | Ensures clean areas for CapCut text in final assembly |
| **Same style suffix** | Ensures consistent art style across all 7 images |
| **B-roll uses different parent** | Anchors = Lola (grandmother), B-roll = younger parent for variety |
| **B-roll student ages 11-13** | Non-negotiable per master directive — varied ages (11, 12, 13) across scenes |
| **Study/academic settings** | Topic-specific: test failure → study desk, floor study, study table |
| **6:19 aspect ratio** | Non-negotiable per master directive |
