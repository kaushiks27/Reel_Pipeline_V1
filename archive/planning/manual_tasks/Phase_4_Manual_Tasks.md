# Phase 4 — Manual Tasks: Image Editing (Canva)

> Touch up all 12 scene images in Canva before Phase 5 (Upscaling).
> This phase is about polishing — not redesigning.

---

## Before You Start

1. Open **Canva** (free account is fine)
2. Create a new project: **Custom size → 768 × 1376 px** (matches your images)
3. Upload all 12 scene images from `assets/scenes/`
4. Work through the edits below scene by scene
5. Export edited images as **PNG** and save back to `assets/scenes/` (overwrite originals)

### Canva Tools You'll Use

| Tool | Where to Find | Use For |
|------|---------------|---------|
| **Magic Eraser** | Click image → Edit Image → Magic Eraser | Removing unwanted text, artifacts |
| **Adjust** | Click image → Edit Image → Adjust | Brightness, contrast, warmth, saturation |
| **Filters** | Click image → Edit Image → Filters | Only if needed for consistency |
| **Text** | Left sidebar → Text | Adding text overlays (habit numbers) |

## Final Consistency Check

After editing, view all 12 images side by side in Canva and check:

- [ ] All images have similar brightness (no jarring jumps between scenes)
- [ ] All images have matching warm color temperature
- [ ] The Speech bubble in Scene 6 is fully removed
- [ ] No AI text artifacts remain in any image
- [ ] The set looks like frames from the same animation

### Quick Fix Guide

| Problem | Fix in Canva |
|---------|-------------|
| One image is too dark | Adjust → Brightness +10 to +15 |
| One image is too cool/blue | Adjust → Warmth +10 to +15 |
| Oversaturated colors | Adjust → Saturation -5 to -10 |
| AI text baked in | Magic Eraser → brush over text → Erase |
| Weird artifact on character | Magic Eraser → brush over artifact → Erase |
