# Animated Prompt Construction — Directive

> Every image prompt in this pipeline must follow the adapted Lesson 22 + Lesson 72 structure, but targeting **animated illustration** instead of photorealism.

## Prompt Structure (6 sections, adapted from Lesson 22)

Every Nanobanana PRO prompt must contain these 6 sections in order:

1. **Subject description** — Who/what, age, ethnicity, expression, character identity
2. **Clothing & accessories** — Culturally appropriate, Filipino-specific details
3. **Setting & environment** — Recognizable Filipino location, warm and bright
4. **Art style & rendering** — Animated style triggers (see below)
5. **Framing & composition** — Camera angle, 9:16 vertical, character positioning
6. **Negatives** — What to EXCLUDE (inverted from Lesson 72)

## Animated Style Triggers (Lesson 72 Categories Adapted)

Use these INSTEAD of the photorealism triggers from Lesson 72:

| Category | ❌ Photorealistic (DO NOT USE) | ✅ Animated (USE THESE) |
|----------|-------------------------------|------------------------|
| 1. Core Style | ultra-realistic, photorealistic, lifelike | soft animated illustration, Pixar-adjacent, warm stylized |
| 2. Camera & Lens | 50mm lens, DSLR, shallow DoF, bokeh | character centered, expressive proportions, eye contact |
| 3. Lighting | soft window light, 5200K, realistic shadows | bright natural lighting, warm glow, soft ambient light |
| 4. Texture & Detail | visible pores, skin micro-details, fabric grain | smooth rounded features, expressive eyes, gentle details |
| 5. Color & Tone | natural color grading, muted earthy tones | warm color palette, pastel and earthy warm tones, vibrant but soft |
| 6. Composition | foreground/background separation | 9:16 vertical, character centered, layered background |
| 7. Quality & Finish | film grain, sensor noise, clean but not polished | clean line work, soft shadows, ambient occlusion |
| 8. Negatives | no cartoon, no CGI, no 3D render | NOT photorealistic, NOT dark, NOT anime, NOT flat 2D |

## Mandatory Inclusions (Every Prompt)

1. **Art style anchor:** "soft 3D animated illustration, Pixar-adjacent" (or similar)
2. **Cultural markers:** Filipino clothing, food, home interiors, or family dynamics
3. **Emotional tone:** Warm, bright, family-friendly, approachable
4. **Safety clause:** "All characters fully clothed in age-appropriate attire. Setting bright and family-friendly. No conflict, no distress."
5. **Format:** "Vertical 9:16 framing"
6. **Negative block:** "NOT photorealistic. NOT dark or cinematic. NOT anime. NOT flat 2D."

## Coursework Grounding Rule

Every prompt in a manual task MUST include a **"How This Prompt Is Built"** section showing:
- Which Lesson 22 structural pattern sections were adapted
- Which Lesson 72 categories were used and what the animated equivalents are
- Any deviations from coursework and why

This ensures traceability back to the source material and prevents prompt drift.
